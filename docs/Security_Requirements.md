# Requirements

## Data Encryption:
- All data transmitted between the client and the server should be encrypted using TLS. 

## Authentication:
- Users must be authenticated before accessing any sensitive endpoints.

## Authorization:
- Implement role-based access control (RBAC) to ensure that only authorized users can perform certain operations.
